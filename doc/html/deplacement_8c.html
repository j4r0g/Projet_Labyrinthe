Description détaillée

Gère les déplacements des fourmis.

Auteur
Boisson Léo, Bussereau Keryann, Ciron Fabien
Version
1.0
Date
14 décembre 2015

Ce module à pour but de gérer les déplacements des fourmis. ainsi que leurs actions lorsqu'elles croisent d'autres insectes elle gère aussi les défloutage des cases.
