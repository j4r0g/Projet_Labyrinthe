Description détaillée

Gère l'interface générale avec l'utilisateur.

Auteur
Boisson Léo, Bussereau Keryann, Ciron Fabien
Version
1.0
Date
14 décembre 2015

Ce module à pour but de demander à l'utilisateur via un menu l'action qu'il souhaite effectuer, le lancement du programme ou la visualisation des règles du jeu.

Documentation des fonctions

int lancement ( )

Lance le jeux avec une génération du labyrinthe, puis un lancement.

Renvoie
Un 0 si l'exécution a réussi, un 1 si elle a été stopée.
int main ( )

Affiche le menu principal et lance soit le lancement soit les règles du jeu, sinon quitte.

Renvoie
Un 0 si l'exécution a réussi.
int regles ( )

Affiche les règles du jeu.

Renvoie
Un 0 si l'exécution a réussi.
int verifvictoire ( t_lab  lab[X][Y])

Gère la vérification d'une victoire, une défaite ou aucun des deux.

Paramètres
lab[x][Y]Reçoie le labyrinthe de taille X, Y.
Renvoie
Un 0 si on a rien, un 1 si on a gagné et un 2 si on a perdu.
